###### Decentralized App Development
> My workflow to create a Dapp begins with writing & deploying a solidity smart contract to a local blockchain using truffle & ganache. In most cases, I unit test the contract using Jest. Next, I model & wire the _backend_ using Express, _frontend_ with React & Web3. I deploy the contract to an Ethereum testnet such as Kovan, Ropsten or Rinkeby using the Remix IDE. Finally, I deploy the smartcontract(s) to Ethereum Mainnet & webapp assets to the InterPlanetary Filesystem (IPFS). I'm experienced in registering mainnet contract sourcecode, abi's and contract profile to Etherscan.
